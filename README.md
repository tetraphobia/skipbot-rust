# Skipbot

A rewrite of the original Skipbot in Rust.

## Requirements

Rust

## Running

```bash
# Clone the repo
git clone https;//github.com/tetraphobia/skipbot-rust.git

cd skipbot-rust

# Copy .env.example to .env, edit it with your Discord token.
cp .env.example .env

# Run the code with cargo
cargo run
```
